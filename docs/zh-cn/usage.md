
# 用法

## preview

> 预览 fetp 所有功能

```bash
$ fetp
```

![](https://makefriends.bs2dl.yy.com/bm1543127896249.gif)

## init

> 构建种子项目

```bash
$ fetp init <模板名称> [项目目录]
# 简写为：
$ fetp i <模板名称> [项目目录]
```

![](https://makefriends.bs2dl.yy.com/bm1543643949529.gif)


## dev

> 开发模式，启动一个Web 服务器

```bash
$ fetp dev
# 简写为：
$ fetp d
```


![](https://makefriends.bs2dl.yy.com/bm1543644512171.gif)
