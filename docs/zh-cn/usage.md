
# 用法

## preview

> 预览 fetp 所有功能

```bash
$ fetp
```


## init

> 构建种子项目

```bash
$ fetp init <模板名称> [项目目录]
# 简写为：
$ fetp i <模板名称> [项目目录]
```

## dev

> 开发模式，启动一个Web 服务器

```bash
$ fetp dev
# 简写为：
$ fetp d
```

## watch

> watch 模式

```bash
$ fetp watch
# 简写为：
$ fetp w
```

## output

> 生产模式，构建并发布到生产环境

```bash
$ fetp output
# 简写为：
$ fetp o
```

## help

> 查看帮助文档

```bash
$ fetp help
# 简写为：
$ fetp -h
```
